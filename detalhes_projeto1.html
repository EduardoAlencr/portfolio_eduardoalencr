Pichuruco - Cofrinho Digital

Sobre o Projeto

Pichuruco é um aplicativo mobile desenvolvido com Flutter que permite casais controlarem suas economias conjuntas. O aplicativo conta com recursos como sincronização de dados em tempo real via Firebase, controle de depósitos, metas de economia, e histórico de transações.

Funcionalidades

  • Sincronização de dados em tempo real entre dispositivos.
  • Criação de metas de economia e monitoramento de progresso.
  • Histórico de depósitos e transferências.
  • Notificações de progresso e lembretes para depósito.

Tecnologias Usadas

  • Flutter
  • Firebase
  • UX/UI Design

Objetivos do Projeto

O objetivo principal do Pichuruco é criar uma ferramenta simples e eficaz para casais controlarem suas finanças de forma colaborativa e organizada. O app tem como foco a usabilidade e acessibilidade para todos os usuários, com uma interface intuitiva.
