Taskfy - Gerenciamento de Tarefas

Sobre o Projeto

Taskfy é um aplicativo de gerenciamento de tarefas desenvolvido com React no frontend e Python/Django no backend. O sistema permite que usuários criem, editem e gerenciem suas tarefas de forma intuitiva e eficiente, com funcionalidades como categorias, prazos e prioridades.

Funcionalidades

  • Criação e edição de tarefas.
  • Categorias e priorização de tarefas.
  • Visualização de tarefas pendentes e concluídas.
  • Integração com banco de dados via Django.

Tecnologias Usadas

  • React
  • Python (Django)
  • CSS (Tailwind CSS)

Objetivos do Projeto

O objetivo do Taskfy é proporcionar uma solução simples e prática para gerenciamento de tarefas, permitindo que os usuários organizem suas atividades diárias de forma eficiente e sem complicação.
